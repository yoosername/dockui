# DockUI CLI

> Interface for managing a DockUI instance from the command line

### Example

```shell
$ dockui --help -vvvv

Usage
    $ dockui <cmd>

  Options
    --help, -h  Show this usage
    --verbosity, -v  Increment the logging verbosity

  Examples
    $ dockui run
    $ dockui apps
    $ dockui apps load <url>

  Info
    Log Level:  debug
```
